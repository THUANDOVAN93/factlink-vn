会社概要を作成するには、まず右袖ログイン画面からログインし、その後左袖のメニューからページ管理に入ってください。

ページ管理画面が表示されます。会社概要、ホーム、内容詳細、プレゼンテーション、お問い合わせの5つの項目に分かれています。

初めてＨＰをご覧になる方は、まず最初に会社概要を作成してください。会社概要を作成しなければ、他のページは作成することができません。

ご希望の言語を選択し、“作成”ボタンをクリックしてください。

テキストボックスに必要事項を記入してください。

+ 会社名
+ 会社スローガン
+ 代表者名、役職
+ 事業内容＆取扱い品目
+ ご住所、電話番号、E-mailアドレス

他、重要事項

+ メニューに表示する名称
通常は「会社情報」とします。ここは初期設定なので自由に入力することができます。

+タイトルバーに表示する名称
通常は「会社名」とします。ここは初期設定なので自由に入力することができます。

+ページ表示順
メニューの表示順序を半角英数字で入力してください。0-10昇順で表示されます。

+ タイトル、会社紹介、画像
○タイトル、会社紹介：会社概要の上部に表示される会社紹介及びそのタイトルについて文章を作成してください。

○画像：会社、工場に関する写真、ロゴをアップロードできます。アップロードできるファイルはJPG、 PNGファイルのみとなっております。

+ 事業内容
会社について、取扱い製品、サービス内容など、いくつかキーワードをご記入ください。

+ タイトル色、追加タイトル色
システムにあるリストからタイトル色、追加タイトルの色を変更することができます。

ご記入し終えましたら、フォーム下の保存ボタンをクリックしてください。

注記
-*マークは記入必須項目です。
- 灰色のテキストボックスは3言語共通項目です。

編集

サンプル写真、会社概要ページを参照してください。日本語コラムにあるアイコンが表示され、クリックすると作成、編集、参照が可能です。 例えば、会社概要をタイ語で作成したい場合は、タイ語コラム(真ん中)にある「編集」ペンシルをクリックしてください。
クリックすると確認ページが参照されます。
クリックすると編集することが可能です。

ページが表示されないようでしたら、キーボード上のＦ5「更新」ボタンを押し、ページを再読み込みしてください。
