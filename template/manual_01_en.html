Rule for creating pages
- Company Profile shall be created first.
- Expand detail is for Content and Presentation only.
- Content Page can create unlimited. Company Profile, Home, Presentation, Inquiry can create only once.
- Any pages except Company Profile can turn on - off.

Basic Member can change main template design of pages by selecting from the list in system. This is for making unique style for each member.
