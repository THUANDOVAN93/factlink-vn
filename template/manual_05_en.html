Presentation page will present data in 3 box, similar to Key panel in Home Page. In this page, you can create row of 3 box more than 1. And you can choose which box is for text and which box is for image.

Presentation cannot create if Company Profile is not available.

Click 'Create' button in the column of language you wish to create.
+ Menu Name
Desire menu name relate to content.

+ Title Bar Name
Your company name is default, but you can change as you desire.

+ Order No.
Fill in the order number of menu that will be shown in any pages. Sort by min-to-max.

+ Title, Detail, Image
You can create paragraph about prologue of follow content. And also upload related picture. Only JPG and PNG file will be allowed.

+ Presentation Box Pattern
Select pattern from the list in system.

Click 'SAVE' button at the bottom of form when you finish.

Remark
- Every textbox that has * in front of label is required information.
- Every textbox that has grey background is global value that will be used for all language.
When you finish, back to Page Management Control Panel. The name of page you just created and some option will be appeared.

Click to add presentation box.

Presentation Box Management split into 2 sections. List Section on the top and Form Section on the bottom.

Each row combine with 3 of presentation box that you can choose which one for text or image

+ Order No.
Fill in the order number of Presentation Box. Sort by min-to-max.These will become to row name.

+ Presentation Box
If you choose Text Box, all field related will be unlocked such as : Title, Title color, Detail
If you choose Image Box, all filed related will be unlocked, the maximum width of image is 220 Pixels, only .jpg and .png are allowed.

Click 'SAVE' button at the bottom of form when you finish. The Detail Column you just saved will be appeared in the List Section.

If you would like back to Page Management Control Panel, please click 'Back' at the top-right corner of Form Section.
